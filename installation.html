Installation Instructions

Follow these steps to install AI Dark Mode Switcher on your Chrome browser

Step 1: Unzip Extension Files

  1. Download the AI Dark Mode Switcher extension ZIP file
  2. Extract/unzip the contents to a folder on your computer
  3. Make sure you have all files: manifest.json, popup.html, popup.js, background.js, styles.css, and icons/ folder
  4. Keep the extracted folder in a safe location for the next steps

Step 2: Open Chrome Extensions

  1. Open Google Chrome browser
  2. Click the three dots (⋮) in the top-right corner
  3. Select "More tools" → "Extensions"
  4. Or type chrome://extensions/ in the address bar

Step 3: Enable Developer Mode

  1. Toggle on "Developer mode" in the top-right corner
  2. You'll see new buttons appear: "Load unpacked", "Pack extension", and "Update"
  3. This enables loading local extensions for development

Step 4: Load Extension

  1. Click the "Load unpacked" button
  2. Navigate to the folder containing your extension files
  3. Select the folder (it should contain manifest.json, popup.html, etc.)
  4. Your extension will be loaded and ready to use!

  • Click the extension icon to open the popup and start using AI Dark Mode Switcher
