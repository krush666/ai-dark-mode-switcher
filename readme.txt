FEATURES:
=========
• Smart Mode Switching: Toggle between Manual and AI modes seamlessly
• AI Mode: Automatic dark mode from 6:30 PM to 8:30 AM using system time
• Manual Mode: User-controlled dark mode schedule with customizable time ranges
• Go Pro Button: Direct link to premium features (configurable URL)
• Privacy-Focused: No external tracking or data collection
• Real-time Updates: Status updates every second to reflect current time
• Persistent Storage: Settings saved across browser sessions
• Context Menu Integration: Quick access and status display
• Visual Feedback: Clear indication of current mode through button styling
• Disabled State Indicators: Time selectors visually indicate when inactive

TECHNICAL SPECIFICATIONS:
========================
• Manifest Version: 3 (Chrome Extension API v3)
• Background Script: Service Worker (Manifest V3 compliant)
• Storage: Chrome Local Storage API
• Permissions: storage, tabs, scripting, contextMenus
• Content Security Policy: Compliant with Chrome Web Store requirements
• Error Handling: Robust messaging with fallback mechanisms
